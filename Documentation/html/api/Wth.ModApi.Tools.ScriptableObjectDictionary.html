Class ScriptableObjectDictionary

This Scriptable Object contains key value pairs for each Scriptable Object used in the game. This is necessary to save references to scriptable objects in the savegame. Use the key in your ISerializable implementation to save/load the reference. The main game and each mod needs an instance of this class.
